Project Description

This project is created for the IT 390-001 (Rapid Development of Scalable Cloud Applications) class in the fall 2024 semester. Teams are tasked with creating a web application following certain criteria. Our group is making a shopping list app that can be used by multiple people in a household to add and remove items from the same list as well as label and sort the list by where the item should be purchased as well as how urgent the item is needed.

Application Features

Primary features in our application are as follows:

  1. Mliti-user functionality
  2. Item tags (location, urgency, etc)
  3. Filtering/sorting by tags
